# VigenereCipherProject

This is a simple C++ program that implements the Vigenere cipher, a classical encryption technique. It allows you to encrypt plainttext messages using a keyword and decrypt ciphertext back to its original message.

---

## Features

- Encrypt plaintext using a keyword
- Decrypt ciphertext using the same keyword
- Supports uppercase alphabetic characters
- Ignores non-alphabetic characters (numbers, spaces, punctuation)
- Demonstrates basic modular arithmetic and sequence processing in C++

---

## Requirements

- C++ compiler (GCC, Clang, or Visual Studio)
